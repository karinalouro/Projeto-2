# Projeto 2 - Easy Shopping Via Mobile

Este projeto é uma página web responsiva que apresenta um layout moderno para um serviço de compras via celular.  
O design é simples e direto, com uma seção de apresentação e outra com navegação e ilustração.

## Tecnologias utilizadas
- **HTML5** para estruturação da página
- **CSS3** para estilização
- **Google Fonts (Oswald)** para tipografia personalizada

## Estrutura do projeto
- `index.html` → Página principal
- `style.css` → Arquivo de estilos
- `Illustration-3-3.png` → Imagem ilustrativa usada no layout

##  Funcionalidades
- Layout dividido em duas colunas:
  - **Esquerda**: título, descrição e botão de ação ("Get Started")
  - **Direita**: menu de navegação e imagem ilustrativa
- Design responsivo para melhor experiência em dispositivos móveis
